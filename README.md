2）生成可执行文件遇到的问题：首先要生成可执行的jar包，由于最初export错误生成jar包而不是可执行jar包，导致失败，后来发现原因后用exe4j软件犯了很多错误，比如search sequence没设置jre路径导致最后失败等，最终成功生成了能运行的exe文件。
